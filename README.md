# Investa – Entrepreneur & Investor Platform

Investa is a full-stack MERN web application that connects entrepreneurs and investors on a single platform. Entrepreneurs can share their business ideas, and investors can explore and show interest in them.

This project is built using React, Node.js, Express, and MongoDB with secure JWT authentication and protected routes for safe user access.

A major highlight of this project is that it was completely developed on an Android mobile device without using a laptop. Tools like Acode, Termux, and API testing apps were used to build and test the entire application.

## Features
- User Signup & Login
- JWT Authentication
- Protected Routes
- Entrepreneur idea posting
- Investor idea browsing and interest system

## Tech Stack
React, Node.js, Express.js, MongoDB, JWT

## Special Highlight
Entire project developed only on Android mobile using Acode and Termux, without any laptop.
